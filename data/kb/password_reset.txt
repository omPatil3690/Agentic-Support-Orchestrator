Title: Password Reset Not Working

Users may report that they are unable to reset their password
or did not receive the password reset email.

Common causes:
- Incorrect email address entered
- Password reset email landed in spam folder
- Account is locked
- Reset link has expired

Steps to resolve:
1. Confirm the user's registered email address
2. Ask the user to check spam or junk folder
3. Generate a new password reset link
4. If the account is locked, unlock it before resetting password
5. Advise the user to use the reset link within the validity period

Important notes:
- Never send passwords in plain text
- Reset links should only be sent to registered email addresses
